# Machine Learning Foundations 2026

This repository tracks my daily progress and code implementations as I build the mathematical and practical foundations for Machine Learning and Computer Vision. 

## Structure
* **`01_NumPy_Drills/`**: Transitioning from standard loops to vectorized operations, including broadcasting and image manipulation.
* **`02_PyTorch_Scratch/`**: From-scratch implementations of fundamental models (Linear Regression, MLPs) without using high-level libraries like scikit-learn.
* **`03_Paper_Summaries/`**: Architecture breakdowns and motivation summaries of foundational CV papers (e.g., AlexNet).
* **`04_Zero_to_Hero/`**: Code and notes following Andrej Karpathy's Zero to Hero lecture series, building neural networks from scratch including micrograd and backpropagation.
